ABOUT THE PROJECT

Tentang HeartGuard AI

HeartGuard AI adalah project Data Science & AI Engineering yang mengaplikasikan Deep Learning untuk membantu deteksi dini penyakit jantung. Sejalan dengan tema Healthy Lives & Well-being.

Metodologi Project

01

Data Collection

Mengumpulkan dataset heart.csv dari UCI Machine Learning Repository (303 sampel, 13 fitur klinis).

02

Exploratory Data Analysis

Analisis distribusi, korelasi antar fitur, dan identifikasi outlier menggunakan visualisasi.

03

Preprocessing

Encoding kategorikal, normalisasi (StandardScaler), dan train-test split (80:20).

04

Model Building (MLP)

Membangun arsitektur Multi-Layer Perceptron dengan TensorFlow Functional API.

05

Training & Validation

Pelatihan dengan custom callback, early stopping, dan monitoring via TensorBoard.

06

Evaluation & Export

Evaluasi accuracy, precision, recall, F1-score; export model ke format .keras.

07

Deployment

Integrasi model ke web dashboard untuk prediksi real-time pengguna.
